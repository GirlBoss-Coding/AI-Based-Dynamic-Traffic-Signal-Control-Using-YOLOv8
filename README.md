# AI-Based-Dynamic-Traffic-Signal-Control-Using-YOLOv8
In fast expanding urban areas like Dhaka, Bangladesh, traffic congestion has emerged as a major problem. Fixed timing cycles are typically used by conventional traffic signal systems, which are unable to adjust to changes in traffic flow in real time. Because of this, these systems frequently result in needless delays, ineffective use of the roads, and higher fuel consumption. This study suggests an AI-powered dynamic traffic signal optimization system that uses deep learning and computer vision to modify signal timing in response to current vehicle density.

The proposed system utilizes YOLOv8-based vehicle detection to identify and classify multiple vehicle types, including cars, rickshaws, buses, trucks, and motorcycles. Traffic video footage from road intersections is processed to detect and count vehicles in each lane. The detected vehicle density is then used as an input for a rule-based traffic signal control algorithm that dynamically adjusts the green signal duration for each lane. Roads with higher vehicle density are allocated longer green signal times, allowing traffic to clear more efficiently while reducing waiting times for vehicles.

In a cloud-based setting, video data processing was used to implement and test the system. The three primary phases of the workflow are adaptive signal timing generation, vehicle detection and density estimation, and video input acquisition. In comparison to conventional fixed-time signal control techniques, the suggested system showed better traffic management through simulation and experimental evaluation.
The AI-based dynamic system can greatly improve traffic flow efficiency, according to performance analysis. Comparing experimental results to traditional signal control strategies, the average vehicle waiting time was reduced by about 10%, and vehicle clearance rates increased by over 14%. These enhancements demonstrate how intelligent traffic management systems can be used to alleviate urban congestion.

Overall, the proposed approach demonstrates how integrating artificial intelligence with traffic infrastructure can promote more intelligent urban mobility. In densely populated cities, the system offers a scalable and affordable solution that may help with sustainable traffic management and increased transportation efficiency.
